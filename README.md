# google-api-wrappers

Lightweight Node.js wrappers for Google APIs, organized as an npm-workspaces monorepo. Each wrapper is published to npm under its own name, but they evolve together here.

## Packages

| Package | npm | What it does |
|---|---|---|
| [`youtube-fast-api`](packages/youtube-fast-api) | [npm](https://www.npmjs.com/package/youtube-fast-api) | Wrapper for the YouTube Data API. |
| [`google-sheets-wizard`](packages/google-sheets-wizard) | [npm](https://www.npmjs.com/package/google-sheets-wizard) | Fetch data from Google Sheets with authentication and error handling. |
